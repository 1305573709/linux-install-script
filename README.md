### 1.1 Ubuntu:
```
cd /tmp
rm ubuntu-installScript
wget https://github.com/cjy37/linux-asp.net-installScript/raw/master/ubuntu-installScript
chmod +x ubuntu-installScript
sudo ./ubuntu-installScript
```
